Инструменты для разработки приложений:
1. Текстовый редактор с подсветкой синтаксиса: 
    -Notepad++;
    -Sublime text;
    -Aditor;
    -Source editor;

2. Компилятор - это программа, которая, исходный код, написанный программистом, преобразует в машинный код, понятный центральному процессору(ЦП, CPU).

3. Отладчик (Debugger) - это программа, которая позволяет пошагово выполнить нашу программу, для того, чтобы найти в ней ошибки на этапе выполнения, и ошибки в агоритме нашей программы.

4. MSDN

Все эти инструменты объединены в 
IDE - Integrated Development Environment/Интегрированная Среда Разработки.
NetBEANS, Eclipse, Code::Blocks, IntelligyIdea.
Visual Studio 2013/2015/2017/2019;
